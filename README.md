# Energy API

This project is a simple backend API built with FastAPI.

## Technologies
- Python
- FastAPI
- Uvicorn

## Features
- Home endpoint
- Energy data endpoint
- Average energy endpoint
- Maximum energy endpoint

## Endpoints
- /
- /energy
- /average
- /max

## Run the project

Install dependencies:

pip install -r requirements.txt

Run the server:

py -m uvicorn main:app --reload
